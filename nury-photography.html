About the project

Nury Photography

A photography website made using several web technologies like HTML, CSS and JavaScript

This website was made as a project for a subject of my engineering degree with the purpose of learning how to use different web technologies.

Each student could create the web site that they wanted and I opted for a photography topic as that was the hobby that I practiced the most at that time.

Nury Photography is composed of different sections: a home page, a photo gallery, blogs, biography, opinions of users, contact information and a login that allows the users that have an account to leave comments and save their favorite photos. In addition, in the last version of the site, a shop section was added, were the users could shop the photos in stock.
