About the project

Starship Run

An obstacle game made with OpenGL and C++

This game was made as a project for a subject of my engineering degree and it was created using only OpenGL and C++.

Spaceship Run consists in reaching the finish line without running out of lives while trying to gain some points along the way. The lives are removed if the spaceship is hit by an asteroid or a wall, and the points can be gained by picking up the red ruby items. However, if the player picks up the shield special item, they will be protected from any damage.

In addition, some other features were implemented, like a first and third person cameras, two different lighting and three speed modes.
